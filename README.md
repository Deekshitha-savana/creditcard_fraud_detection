# creditcard_fraud_detection
# Credit Card Fraud Detection using Machine Learning

## Project Overview

This project detects fraudulent credit card transactions using Machine Learning. It uses the Credit Card Fraud Detection dataset and a Random Forest Classifier to classify transactions as Fraud or Genuine.

## Features

- Load dataset from URL
- Data exploration
- Missing value analysis
- Class distribution visualization
- Correlation heatmap
- Random Forest Classifier
- Model evaluation
- Confusion Matrix
- Feature Importance visualization
- Predict whether a transaction is Fraud or Genuine

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset

The dataset contains:

- 284,807 transactions
- 31 columns
- 492 fraudulent transactions

Target column:

- Class = 0 → Genuine
- Class = 1 → Fraud

Dataset URL:

https://raw.githubusercontent.com/nsethi31/Kaggle-Data-Credit-Card-Fraud-Detection/master/creditcard.csv

## Machine Learning Algorithm

Random Forest Classifier

## Output

- Dataset Summary
- Fraud vs Genuine Graph
- Correlation Heatmap
- Classification Report
- Confusion Matrix
- Feature Importance
- Prediction Result

## Accuracy

Approximately 99.9% (Random Forest)
